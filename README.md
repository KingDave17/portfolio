# 🚀 Interactive Portfolio | David Ajibade

A high-fidelity, interactive web-based portfolio showcasing full-stack mobile and web development skills. Built to provide recruiters with a **"Show, Don't Tell"** experience through live project simulators.

🔗 **Live Demo:** [ajidavidportfolio.vercel.app](https://ajidavidportfolio.vercel.app)

---

## ✨ Key Features

### 📱 Interactive Project Simulators
Instead of static screenshots, this portfolio features live mobile and web suite simulators.
- **TouringApp Demo**: A 1:1 mobile companion simulator built with React Native, showing real-time location services and biometric authentication workflows.
- **Web Platforms**: Full-stack demonstrations for hospitality and management systems.

### 💼 Professional Design
- **Modern UI/UX**: Clean dark-mode aesthetic with glassmorphism and smooth CSS animations.
- **SEO Optimized**: Fully configured meta tags for LinkedIn/Twitter previews and search engine visibility.
- **Mobile Responsive**: Seamless experience across all devices.
- **Integrated Contact Form**: Powered by Web3Forms for direct-to-inbox lead generation.

---

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, Vanilla CSS3 (Custom Design System)
- **Deployment:** Vercel (CI/CD Pipeline)
- **APIs & Tools:** Web3Forms API, Lucide React (Icons), Git/GitHub

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KingDave17/portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
